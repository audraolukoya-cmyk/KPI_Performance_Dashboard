# KPI Performance Dashboard

## Overview
An interactive data visualization project designed to track core competency metrics, support data-driven leadership decisions, and enable real-time performance monitoring.

## Objective
To develop a dynamic tracking system monitoring baseline performance metrics for **400 active stakeholders**. The primary goal was to isolate performance gaps in core competencies and implement data-driven strategies to ensure a higher rate of benchmark achievement.

## Tools Used
* **Data Processing & Visualization:** Excel / Power BI
* **Data Modeling:** Power Query 

## Process
* **Data Preparation:** Cleaned, transformed, and structured raw datasets for analysis, resolving null values and formatting inconsistencies.
* **Metric Development:** Engineered custom KPI metrics to measure success against baseline targets accurately.
* **Dashboard Design:** Built an intuitive, user-friendly interface with dynamic visualizations to replace static reporting.
* **Analysis:** Interpreted data patterns to highlight critical performance variance and isolate individuals failing to meet baseline standards.

## Dashboard Features & Visualizations
To ensure complex metrics were easily digestible for leadership, the dashboard was structured with the following interactive elements:

* **Executive Summary Cards (KPIs):** Positioned at the top to give decision-makers an immediate, high-level view of total stakeholders, overall proficiency rates, and benchmark averages.
* **Segmented Performance Chart:** A stacked column visual that breaks down achievement by specific categories, allowing for quick comparison across groups.
* **Interactive Slicers & Filters:** Dropdown menus that empower users to dynamically drill down into the data by specific dates, categories, or performance tiers.
* **Drill-Through Data Table:** A detailed, sortable grid at the bottom of the canvas providing row-level transparency on individual stakeholder performance.
* **Left-Rail Navigation Pane:** A clean vertical menu structure that mimics professional enterprise software for a seamless user experience.

## Key Insights
* Uncovered specific performance trends across measured intervals to track growth.
* Pinpointed exact sub-groups requiring targeted resource allocation to maximize proficiency.
* Streamlined executive decision-making by delivering visual summaries that immediately highlight compliance and achievement gaps.

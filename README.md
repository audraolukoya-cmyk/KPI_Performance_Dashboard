# KPI Performance Dashboard

## Overview
This project visualizes key performance indicators (KPIs) to support data-driven decision-making and performance monitoring.

## Objective
To track trends, identify performance gaps, and provide actionable insights through a dashboard.

## Tools Used
- Excel / Power BI

## Process
- Cleaned and structured dataset
- Created KPI metrics
- Built dashboard with visualizations
- Analyzed trends and performance patterns

## Key Insights
- Identified performance trends over time
- Highlighted areas of improvement
- Enabled quick decision-making through visual summaries
